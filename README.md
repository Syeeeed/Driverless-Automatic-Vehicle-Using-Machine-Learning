## Overview
This project focuses on developing a driverless vehicle system using machine learning for perception and decision-making. It integrates computer vision and reinforcement learning to enable autonomous navigation in dynamic environments.

---

## Objective
- Enable real-time autonomous driving  
- Detect and interpret road features  
- Learn adaptive path planning behavior  

---

---

## Key Components

### 1. Perception (Computer Vision)
- Convolutional Neural Networks (CNNs)  
- Object detection (vehicles, obstacles)  
- Lane detection and tracking  
- Traffic sign recognition  

---

### 2. Decision Making
- Reinforcement Learning (RL)  
- Adaptive path planning  
- Environment-based action selection  

---

### 3. Control System
- Steering control  
- Speed regulation  
- Real-time response to predictions  

---


---

## Evaluation Metrics
- Accuracy (object detection)  
- Precision / Recall  
- Mean Average Precision (mAP)  
- Navigation performance (path efficiency)  

---

## Tech Stack
- **Languages:** Python  
- **ML/DL:** TensorFlow / PyTorch  
- **Vision:** OpenCV  
- **Learning:** CNN, Reinforcement Learning  
- **Environment:** Jupyter Notebook  

---

## Current Progress
- CNN-based perception model implemented  
- Initial RL-based navigation model developed  
- Simulation-based testing completed  

---

## Ongoing Advancement
Further development is being conducted using the **CARLA Simulator** to model diverse real-world driving scenarios. The system is being extended with:
- Simulation of complex environments (urban, traffic, weather conditions)  
- Integration of appropriate sensor setups  
- **LiDAR-based navigation** for improved perception and spatial awareness  

---

## Future Work
- Real-world testing integration  
- Improve model accuracy and robustness  
- Multi-sensor fusion (camera + LiDAR + ultrasonic)  
- End-to-end autonomous pipeline optimization  

---

## Vision
To develop a fully autonomous, intelligent vehicle capable of safe and adaptive navigation using AI-driven perception and decision-making systems.
